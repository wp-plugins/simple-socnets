This plugin was built by the Maine WordPress Meetup group to make it really easy to add social network icons to your posts.


== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.  After activation, you MUST visit the settings page (under "Settings > Simple Socnets") and press "Save"
